EEG Headset

  • This software is designed to operate the Puzzlebox Orbit helicopter using a NeuroSky MindWave Mobile EEG
  • An EEG measures electrical activity in the brain on the microvolt scale (millionths of a volt)
  • Signal processing algorithms convert these readings into measurements of mental clarity and concentration
  • This software treats those detections as a trigger to transmit control commands to the Orbit helicopter
