Steering

Steer the Orbit using your device's tilt sensors
  • While it is possible to steer an Orbit helicopter using brainwaves via EEG, this requires advanced hardware with multiple electrodes, whereas the NeuroSky MindWave Mobile only has a single channel
  • As an alternative it is also possible to steer the Orbit while in flight by using tilt sensors found in many mobile devices.
  • It is recommended to use focused concentration or meditation to achieve hovering, then steer using this feature
